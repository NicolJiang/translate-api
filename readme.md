# Google Translate Server
A simple web interface for Google Translate.

# Interface

```json
Get Request:
/?w=word

Response:
{
    "word": "word",
    "text": "字",
    "candidate": [
        "字",
        "词",
        "单词",
        "话",
        "言",
        "言辞",
        "笔墨",
        "约言",
        "字",
        "词",
        "单词"
    ]
}

```
